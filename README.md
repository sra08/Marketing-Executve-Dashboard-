# 📊 Marketing Performance Dashboard

An interactive **Looker Studio dashboard** built to help marketing teams monitor campaign performance across multiple advertising platforms through real-time KPI tracking and executive reporting.

🔗 **Live Dashboard:** [View Dashboard](https://datastudio.google.com/reporting/d9c36517-d948-4252-9bf3-1ec8ccaf0ba3)

---

## Overview

This dashboard consolidates marketing data into a single reporting interface, allowing stakeholders to compare campaign performance, monitor advertising spend, and evaluate business outcomes without manual reporting.

---

##  Features

- Executive KPI Dashboard
- Campaign Performance Analysis
- Platform Comparison (Google Ads, Meta & TikTok)
- Interactive Date Filters
- ROAS, CPA & CTR Tracking
- Revenue & Spend Trends
- Budget Allocation Analysis
- Responsive Dashboard Design

---

## Key Metrics

- Revenue
- Spend
- ROAS
- CPA
- CTR
- Conversions
- Impressions

---

## 🛠 Tech Stack

- Looker Studio
- Google Ads
- Meta Ads
- TikTok Ads
- Google Sheets
